# Rift

**A local-workspace coding assistant for Windows.** Point Rift at a folder and chat with Claude — it reads, searches, edits, and runs git in place, all on your machine. Built on Tauri 2 + SvelteKit + Rust.

### Download

Grab the latest **`Rift-win-Setup.exe`** from the [**latest release**](https://github.com/Blazzer10200/rift-releases/releases/latest). Per-user install, no admin required — and Rift self-updates from here on (Velopack), so you only install once.

> On first run, SmartScreen may warn (no code-signing cert yet) → **More info → Run anyway**.
